# Carpeta de Calidad - Red Show MVP

Esta carpeta centraliza todos los reportes, evidencias y registros de calidad del proyecto Red Show.

## Estructura

- **reports/revision-tecnica/** - Reportes de revisión técnica por fecha
- **reports/revision-tecnica/evidencias/** - Screenshots y logs de hallazgos
- **reports/revision-tecnica/EVIDENCIAS.md** - Índice de evidencias

## Ubicación de Reportes

Los reportes se organizan cronológicamente: `revision-tecnica/YYYY-MM-DD/`

Cada reporte incluye:
- Módulo evaluado
- Hallazgos (críticos, mayores, menores)
- Recomendaciones
- Vinculación con issues
